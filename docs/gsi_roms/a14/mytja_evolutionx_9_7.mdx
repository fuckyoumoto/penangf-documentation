    ### Keyboard
    **Problem:** The keyboard doesn't work

    **Solution:**
    - Install [SimpleKeyboard](https://github.com/rkkr/simple-keyboard/releases/latest) via adb
    - Open the SimpleKeyboard app and in the settings under "On-screen keyboard" uncheck Gboard and select Simple Keyboard
    - Change the voice assistant to the SimpleKeyboard input (bottom right, keyboard icon) when trying to activate it


    ### Brightness
    **Problem:** Brightness is not adjusted smoothly and at large intervals

    **Solution:** Phh Treble Settings -> Misc Features -> Force alternative backlight scale
